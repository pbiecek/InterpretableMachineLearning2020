Tak jak w poprzednich zadaniach domowych pracujemy na zbiorze danych HELOC. Wytrenujmy na nich model XGBoost. Jak wida z poniszej tabelki najbardziej znaczca jest zmienna ExternalRiskEstimate. Jest to zgodne z poprzednimi dowiadczeniami, poniewa sam t zmienn mona do dobrze wytumaczy wynik (accuracy 0.70). Zauwamy rwnie jak bardzo ta zmienna dominuje nad pozostaymi. Waga drugiej w kolejnoci zmiennej jest blisko 9-krotnie mniejsza ni pierwszej. Widzimy te e 6 zmiennych ma niedodatni warto, co znaczy e po usuniciu nich powinnimy otrzyma lepszy model.

def avg(model, x, y, eq):
    pred = model.predict(x)
    acc = [1 if eq(e, s) else 0 for (e,s) in zip(pred, y)]
    return np.average(acc)

def filter_diff(x):
    rp = x['RiskPerformance']
    dis = len(set(rp))
    return dis == 1

def get_data():
    read = pd.read_csv("heloc_dataset_v1.csv")
    data_columns = [f for f in read.columns if f != 'RiskPerformance']
    grouped = read.groupby(data_columns)
    filtered = grouped.filter(lambda x: True)
    return filtered['RiskPerformance'], filtered[data_columns]

def prep_data():
    res, data = get_data()
    res = [1 if t == "Good" else 0 for t in res]
    for c in data.columns:
          data[c] = [-9 if t == -7 or t == -8 or t == -9 else t for t in data[c]]
    orig_data = data
    return orig_data, np.array(res), pd.DataFrame(data, columns=orig_data.columns)

Z powyszych wynikw wida e usunicie dominujcej zmiennej sprawio e wicej zmiennych zaczo mie wiksze znaczenie, takie jak NetFractionRevolvingBurden, AverageMInFile czy MSinceMostRecentInqexcl7days. Wynika z tego e usunicie tej zmiennej pozytywnie wpywa na wyjanialno, zwaszcza e opierajc si na wiedzy biznesowej, warto zmiennej ExternalRiskEstimate musi zalee od tych zmiennych. Zmienne te pojawiay si rwnie w poprzednich pracach jako istotne, co potwierdza poprawno wyjanienia. Pomimo podobiestw wystpuj te rnice midzy powyszymi modelami. W modelu XGBoost drug z najwaniejszych zmiennych jest NumBank2Natl(...), podczas gdy w modelach lasu losowego ta zmienna jest odpowiednio na 6-tej i 10-tej pozycji.
